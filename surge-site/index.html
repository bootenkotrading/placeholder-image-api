Usage

Code Examples

Integrate in seconds — use it anywhere you'd use an image URL

<!-- Simple placeholder -->
<img src="http://149.202.58.157:8892/400x300"
     alt="Placeholder">

<!-- Custom colors & text -->
<img src="http://149.202.58.157:8892/800x600/1a1a2e/e94560?text=Hero+Image"
     alt="Hero placeholder">

<!-- Specific format -->
<img src="http://149.202.58.157:8892/400x300.webp"
     alt="WebP placeholder">
/* Background placeholder */
.hero {
  background-image: url("http://149.202.58.157:8892/1920x1080/0a0a0a/00d2ff?text=Hero");
  background-size: cover;
}

/* Profile avatar placeholder */
.avatar {
  background-image: url("http://149.202.58.157:8892/128x128/2d3436/6c5ce7?text=Avatar");
  border-radius: 50%;
}

/* Card thumbnail */
.card-thumb {
  background-image: url("http://149.202.58.157:8892/350x250/1a1a2e/e94560?text=Card");
}
// Fetch placeholder as blob
const response = await fetch("http://149.202.58.157:8892/400x300");
const blob = await response.blob();
const url = URL.createObjectURL(blob);
document.querySelector("img").src = url;

// Dynamic placeholder generation
function placeholder(w, h, bg="0a0a0a", fg="00d2ff", text="") {
  const t = text ? `?text=${encodeURIComponent(text)}` : "";
  return `http://149.202.58.157:8892/${w}x${h}/${bg}/${fg}${t}`;
}

// Set image src dynamically
document.getElementById("thumb").src =
  placeholder(800, 600, "1a1a2e", "e94560", "Featured");
# Using requests
import requests

url = "http://149.202.58.157:8892/400x300/0a0a0a/00d2ff?text=Hello"
response = requests.get(url)

# Save to file
with open("placeholder.png", "wb") as f:
    f.write(response.content)

# Using PIL with BytesIO
from io import BytesIO
from PIL import Image

resp = requests.get("http://149.202.58.157:8892/400x300")
img = Image.open(BytesIO(resp.content))
img.show()

# Bulk generate placeholders
sizes = [(800,600), (400,300), (200,200)]
for w, h in sizes:
    r = requests.get(f"http://149.202.58.157:8892/{w}x{h}")
    with open(f"placeholder_{w}x{h}.png", "wb") as f:
        f.write(r.content)
# Download a placeholder image
curl -o placeholder.png "http://149.202.58.157:8892/400x300"

# Custom colors and text
curl -o hero.png "http://149.202.58.157:8892/1920x1080/0a0a0a/00d2ff?text=Hero"

# JPEG format
curl -o photo.jpg "http://149.202.58.157:8892/800x600.jpg/1a1a2e/e94560?text=Photo"

# WebP format
curl -o image.webp "http://149.202.58.157:8892/400x300.webp"

# With API key (Pro/Enterprise)
curl -H "X-API-Key: your-api-key" \
     -o placeholder.png \
     "http://149.202.58.157:8892/400x300"
Documentation

API Reference

Simple, RESTful — generate images with a single GET request

GET /{width}x{height}/{bg_color}/{fg_color}?text=&format=

Generate a placeholder image with custom dimensions, colors, text, and format.

ParameterTypeDescription
widthintegerImage width in pixels (1–4000)
heightintegerImage height in pixels (1–4000)
bg_colorstringBackground color as 6-digit hex (no #). Default: 0a0a0a
fg_colorstringText color as 6-digit hex (no #). Default: 00d2ff
textstringCustom text overlay. Default: shows dimensions
formatstringImage format: png (default), jpg, webp, svg. Also settable via file extension in the path.
GET /{width}x{height}

Shorthand — generates a placeholder with default dark background and accent text color. Text shows dimensions.

GET /{width}x{height}.{format}

Specify output format via file extension. Supports png, jpg, webp, svg.

GET /health

Health check endpoint. Returns {"status": "ok"} when the service is running.

GET /docs

Interactive Swagger/OpenAPI documentation (available on the API server directly).
