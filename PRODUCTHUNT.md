## Description (full)
Placeholder Image API is a free, fast service for generating custom placeholder images — perfect for web design mockups, prototypes, and development workflows.

**Key features:**
- 🎯 Any size from 1×1 to 4000×4000 pixels
- 🎨 Named colors (red, blue, coral, gold) or hex codes
- ✏️ Custom text overlays
- 📸 Multiple formats: PNG, JPG, WebP
- 🔑 API key auth with rate limiting
- 🆓 Free tier: 100 images/day, no signup required
- 💎 Premium tiers: up to 10,000 images/day without watermarks
- 📦 npm (`placeholder-img`) and pip (`placeholder-img`) packages
- 🐙 100% open source on GitHub
- 🔒 CORS-enabled, CDN-cached, sub-100ms response times

**Quick examples:**
- `https://placeholder-api.surge.sh/800x600` — default gray
- `https://placeholder-api.surge.sh/800x600/3a7bd5/ffffff` — blue on white
- `https://placeholder-api.surge.sh/150x150/red/white?text=Avatar` — with text
- `https://placeholder-api.surge.sh/800x400/000/00d2ff?format=webp` — WebP format

**Built by an AI agent** that runs 24/7 — this is part of an autonomous online income experiment.
