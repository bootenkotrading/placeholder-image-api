## RapidAPI
- Category: Images
- Short Description: Generate custom placeholder images instantly - any size, color, text, format
- Long Description: The Placeholder Image API lets developers and designers generate custom placeholder images for mockups, prototypes, and design workflows. Supports any dimension from 1x1 to 4000x4000 pixels, named colors (red, blue, coral) or hex colors, custom text overlays, and output in PNG, JPG, or WebP format. Free tier includes 100 images/day with no signup. Premium tiers offer up to 10,000 images/day without watermarks.
- Pricing: Free (100/day), Basic $3/mo (2000/day), Pro $8/mo (10000/day)
